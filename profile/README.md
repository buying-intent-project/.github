# Buying-Intent Classifier
## Overview

This repository delivers a production-focused buying-intent classifier for an e-commerce platform. It supplies controlled dataset structure, reproducible benchmarks, and deployment-ready model packaging. The objective is stable, explainable intent scoring across product interactions.

## Mission

Provide a unified system for:
- Structured text-event datasets tied to user actions
- Reproducible evaluation across classifier variants
- Deployment artifacts suitable for real-time scoring on an e-commerce stack
